# Sentry


## sentry-errors
Shows a list of sentry errors

### Config
```json
{
  "id": "",
  "widget": "sentry-errors",
  "config": {
    "title": "Errors",
    "app": "APP",
    "projectId": "PROJECT_ID"
  }
}
```

---

## sentry-issues
Shows a list of sentry issues

### Config
```json
{
  "id": "",
  "widget": "sentry-issues",
  "config": {
    "title": "Issues",
    "app": "APP",
    "projectId": "PROJECT_ID"
  }
}
```
